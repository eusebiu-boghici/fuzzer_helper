# Fuzzer Command Line Builder
check live page here - https://eusebiu-boghici.github.io/fuzzer_helper/

or host it yourself
```sh
git clone https://github.com/eusebiu-boghici/fuzzer_helper
cd fuzzer_helper
python -m http.server &
firefox http://localhost:8000
```
